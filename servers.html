The Blade Chassis has one DVD R/W Drive.

Layer 2 Ethernet switches for connecting all the blades in the chassis to the external LAN network in redundant manner. Ethernet switches support Two I/O Ports per blade and has Six 10/100/1000 Mbps copper connected external ports.

Chassis has Dual Redundant SAS Storage Controller. Controller Modules can connect External storage with Chassis in fully redundant manner.

IP Based KVM functionality for internally/externally Module Management for all the blade servers. Dual redundant Management modules can be configured for the Management of Blade & I/O Switches on the chassis.

N+N Redundancy on the Power supplies powering the chassis. Configured blades have redundant connectivity for Power to the Mid plane.

Cooling is provided for all the components in completely redundant manner using Hot Swap Blowers/High Speed fans
